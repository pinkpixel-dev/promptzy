## What is Promptzy?

**Promptzy** is a sleek, modern web app for organizing and managing your AI prompts. Tag them, search them, sync them to the cloud, and generate new ones with an integrated AI assistant — all from a single, beautiful interface.

<CardGrid stagger>
  <Card title="Organize Everything" icon="list-format">
    Categorize prompts as **System**, **Task**, **Image**, or **Video**. Add
    custom tags and find exactly what you need in seconds.
  </Card>
  <Card title="Cloud Sync" icon="cloud-upload">
    Supabase-backed storage keeps your prompt library in sync across every
    device — phone, tablet, desktop.
  </Card>
  <Card title="AI Assistant" icon="rocket">
    Generate ready-to-use prompts with one click using the built-in AI
    assistant, powered by [Pollinations](https://pollinations.ai).
  </Card>
  <Card title="Install Anywhere" icon="laptop">
    Run as a PWA, native Linux desktop app, Docker container, or straight from
    npm — your choice.
  </Card>
</CardGrid>
